# Flutter UI Practice

Collection of small Flutter projects focused on UI.

## What you will find

* Multiple UI screens
* Reusable components
* Simple layouts for practice

## Goal

* Improve UI building skills
* Practice screen design
* Create reusable widgets

## Structure

Each folder contains a separate UI project.

Examples:

* auth_ui
* onboarding_ui
* profile_ui

## Notes

* UI only
* No backend
* Focus on clean structure
